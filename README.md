## ğŸ”‘ Available Data Fields

### Race Overview
- Track name, state, country
- Distance (meters)
- Track condition (Good4, Good, Soft, Heavy)
- Weather conditions
- Race type (T=Thoroughbred, G=Greyhound)
- Expert race comments

### Runner Information
- **Odds**: `odds.fixed_win`, `odds.fixed_place`
- **Odds Movements**: `flucs_with_timestamp` (opening, high, low, recent)
- **Form**: `last_twenty_starts` (1-9 for placings, x=unplaced)
- **Best Time**: `best_time` (greyhounds)
- **Speedmap**: `speedmap.label` (Front Runner, Midfield, Back Marker)
- **Form Indicators**: "Nearing Another Win", "Form on Good", etc.
- **Jockey/Trainer**: Names, locations, past performance stats
- **Barrier/Box**: Draw positions
- **Weight**: Allocated and total (horses)
- **Gear**: Blinkers, tongue ties, etc. (horses)
- **Scratched Status**: `is_scratched` flag
